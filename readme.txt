== Description ==

'SyntaxHighlighter TinyMCE Button' provides additional button for 'Visual Editor(TinyMCE)' and will help to type `<pre>` tag for SyntaxHighlighter.

This plugin is based on 'CodeColorer TinyMCE Button'.

'Allow to type tabs' option allow you to type tab in 'Visual Editor' to indent.
You can configure this option through Setting -> SH TinyMCE Button.

Localization: English(Default), Japanese

= Notes =

This plugin is designed to work with a plugin based on Alex Gorbatchev's SyntaxHighlighter Ver, 2.0 or higher.

You can not use this plugin with some 'SyntaxHighlighter'-based plugins which do not support `<pre>` tag.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. If you need, go to Settings -> SH TinyMCE Button to configure.
1. Insert your code to your post and markup your code to use 'pre' button.
